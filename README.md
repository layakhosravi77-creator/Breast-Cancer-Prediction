# 🩺 Breast Cancer Prediction Using Machine Learning Classification Techniques

## 📌 Overview

This project focuses on developing a **binary classification model** to predict the presence or absence of breast cancer using machine learning techniques. The dataset contains patient-level information, including demographic characteristics, reproductive history, hormonal factors, medical history, and clinical findings associated with breast cancer risk.

The project covers the complete machine learning workflow, from data preprocessing and exploratory data analysis to model training, evaluation, and comparison.

---

## 🎯 Objectives

- Predict breast cancer diagnosis using supervised machine learning algorithms.
- Perform comprehensive data preprocessing and feature engineering.
- Compare the performance of multiple classification models.
- Evaluate models using standard classification metrics.

---

## 📂 Dataset

The dataset includes various patient-related features such as:

- Demographic information
- Reproductive history
- Hormonal factors
- Medical history
- Clinical examination findings

**Target Variable**

- **0** → No Breast Cancer
- **1** → Breast Cancer

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/Breast-Cancer-Prediction.git
```

Navigate to the project folder:

```bash
cd Breast-Cancer-Prediction
```

Install the required packages:

```bash
pip install -r requirements.txt
```

## ▶️ Usage
